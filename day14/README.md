The program calculates hotel bills for multiple customers based on room type,number of days,and optional services (breakfast,Wi-Fi,gym).
It assigns room rates (₹2500,₹4000,₹6000) and adds extra charges for selected services.
The total cost is computed,with a 10% discount applied if the total exceeds ₹10,000.
GST of 12% is added to the final bill for each customer.
Finnaly, it  displays each customer's final bill along with total revenue,highest, and lowest bills.
